## mdx博客书写语法
元信息

```bash
---
title: "GitHub Actions 部署 nextjs 博客"
date: "2026-03-10"
tags: ["部署", "GitHub Actions", "CI/CD"]
---
```

二级标题

```bash
##
```

无序列表

```bash
- 无序列表
```

有序列表

```bash
1. 有序列表
2. 
```

代码块

```bash
​```bash
代码块
```
```

插图

​```bash
![提示词](链接)
```

引用

```bash
> 引用
```

斜体

```bash
*斜体*
```

删除线

```bash
~~x~~
```

粗体

```bash
** 粗体 **
```

关键词

```bash
<span className="keyword">核心概念</span>
```

引用链接

```bash
[链接名](链接地址)
```

行内代码

```bash
`行内代码`
```
